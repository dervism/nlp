# Python NLP testing app

### Setup
Start virtual environment:

```
source ./venv/bin/activate
```

Install dependencies:

```
pip3 install --upgrade pip setuptools tensorflow torch torchvision transformers openpyxl stanza
```

Run the program:

```
python nlp.sa.py
```
